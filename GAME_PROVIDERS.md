## Integration Process

### **For API-Based Providers**:

1. **Get API Credentials**:
   - Sign up with provider
   - Get API key/secret
   - Get API documentation

2. **Add to Your Site**:
   - Use Admin Panel â†’ "Add New Game"
   - Enter game URL from provider
   - Enter thumbnail URL
   - Fill in game details

3. **Test Integration**:
   - Click "Play Now"
   - Verify game loads
   - Test balance updates
   - Verify transactions log

### **For Purchased Games**:

1. **Download Game Files**:
   - Get HTML5 game files
   - Get game assets (images, sounds)

2. **Upload to Your Hosting**:
   - Create `/games/` folder
   - Upload game files
   - Set up game URL

3. **Add via Admin Panel**:
   - Game URL: `https://yoursite.com/games/game-name/index.html`
   - Add thumbnail
   - Fill in details

4. **Test Integration**:
   - Click "Play Now"
   - Verify game works
   - Test all features
