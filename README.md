## üöÄ Getting Started

### **Prerequisites**
- Node.js (version 14 or higher)
- npm or yarn

### **Installation**
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd casino-max
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:3000`

### **Building for Production**
```bash
npm run build
```
